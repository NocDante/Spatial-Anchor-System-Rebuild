# Spatial Anchor System Rebuild
[繁體中文] | [English](README_English.md)

---

以前開發 XR 專案時，雖然 Meta 的 SDK 拿來接一接功能就動了，但老實說，我對底層到底在跑什麼其實有點似懂非懂。

所以這次我打算將之前做過的東西延伸重練。

## 此專案的目標
1. 更通用的架構：以後設計其他專案時，擁有一套快速架設且正確的思考方式。

2. 邏輯拆解與 SOLID：嘗試使用 SOLID 原則與 Design Pattern，結合過往經驗寫出具理論基礎的程式。

3. 熟悉新版 SDK：深入掌握更新後的 Meta SDK。

---

## 小結

我想透過這次重寫，把 Unity 的開發直覺磨利一點，建立起一套自己也認同的開發節奏。
